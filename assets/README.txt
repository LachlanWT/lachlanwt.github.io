Put your media files in this folder.

Expected files:
  profile.jpg   -> your sidebar photo (any square-ish image works)
  resume.pdf    -> shown on the Resume tab
  project-1.jpg -> optional images for project pages, etc.

After adding a file here, reference it from the HTML, e.g.:
  <img src="assets/profile.jpg">        (from a top-level page)
  <img src="../assets/profile.jpg">     (from a page in projects/)

You can delete this note file once you've added your own.
