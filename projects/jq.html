Overview

This was the final project for CS 61B (Data Structures). Build Your Own World is a Java engine that procedurally generates and renders an explorable 2D tile world. From a random seed, the generator lays out pseudorandomly sized and placed rooms and connects them with hallways into a single, fully connected world, and because it is seed-driven, the same seed always reproduces the same map.

What I built

On top of the world generator I built an interactive engine with keyboard-driven movement, a heads-up display, and the ability to save and load the game state so a world can be quit and later resumed exactly where it left off. The project brought together procedural generation, data structures, and interactive rendering into a single application.
